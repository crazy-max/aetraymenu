AESTAN TRAY MENU - EXAMPLES - AEONSERV CONTROLLER
=================================================

This directory contains the configuration of the Aeonserv Controller application. It shows you how a real-world software package uses AeTrayMenu.

Aeonserv is a package for Windows that includes the Apache http server, MySQL, PHP, Perl and well-known scripts like phpMyAdmin, awStats and the GeekLog portal system. It includes an easy installer that lets you turn your computer into a full-blown webserver in a matter of minutes. And best of all, it's completely free.
See http://aeonserv.com for more information.
