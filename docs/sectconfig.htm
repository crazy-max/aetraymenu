[Config] Section

  
This section contains general settings, some of which are required. Here is an example of a [Config] section: [Config] ImageList=images.bmp TrayIcon=trayicon.ico ServiceCheckInterval=3 ID={42F9F6BD-49C0-42E6-8250-25C98ACE1DAB}

The following directives can be placed in the [Config] section:

General

  • ID
  • ImageListREQ
  • TrayIconREQ

Services-related

  • ServiceCheckInterval
  • ServiceGlyphPaused, ServiceGlyphRunning, ServiceGlyphStopped
  • TrayIconAllRunning, TrayIconNoneRunning, TrayIconSomeRunning
